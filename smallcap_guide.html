3. L'Opportunity Score

Score = 0.25·Sous-éval + 0.20·Catalyseur + 0.15·Float/Squeeze
    + 0.15·Momentum + 0.10·Survie + 0.15·Thème
    × multiplicateur thématique (1.0 → 1.5)
ComposanteCe qu'elle mesureSource (gratuite)
Sous-évalPas déjà envolée (marge de hausse)Prix Yahoo (proxy)
CatalyseurFiling récent + direction (8-K↑ / 424B5↓)EDGAR
Float / SqueezeFlottant faible = explosifFMP (shares)
MomentumRVOL du titre + momentum du thèmeFMP + Yahoo
SurvieCapacité à tenir (proxy cap, pénalité dilution)FMP

★ potentiel élevé = score > 85 + thème chaud (heat ≥ 70) + dernier catalyseur haussier. Statuts : CONVICTION (≥85) · RADAR HOT (≥78) · SUIVI (≥60) · ÉVITER (<60).
Exemple réel : OKLO score 46 ÉVITER — son dernier filing est un 424B5 (dilution) et sa cap a explosé (~11 B, plus un small-cap). Le robot te montre la réalité, pas le narratif bullish.

6. Repérer les IPO & lock-up sur Finviz

Les IPO récentes = float faible = mouvements violents. La méthode :

1
Screener → Filters → IPO Date → « In the last year » (ou « last quarter » / « last month » pour le frais).
2
Combine avec Market Cap = Small/Micro + Float = Under 20M → IPO récentes à flottant faible.
3
Trie par IPO Date (Order by) pour voir les plus récentes en haut.
Le lock-up = ton catalyseur DATABLE. ~180 jours après l'IPO, les initiés peuvent vendre → pression vendeuse prévisible. Note la date d'IPO + 180j : c'est un short / une zone à éviter en long. Le module calcule ça automatiquement (champ lockupDate).
Avant même le ticker : le S-1 (sur EDGAR) est déposé avant l'IPO — c'est là qu'on repère une future cotation, bien avant Finviz.

La chaîne complète, d'un filing EDGAR jusqu'à ton écran :

2. EDGAR full-text l'indexe    → minutes → ~24h
3. Le Schedule n8n se déclenche → 0 → 24h d'attente  ◀ le maillon qui domine
4. Publish → GitHub Pages     → ~1 min
5. Tu ouvres la page (Ctrl+Shift+R) → instant

10. Gestion de sortie

« Savoir entrer, c'est 30 %. Savoir sortir, c'est 70 %. » La sortie garantit la pérennité sur le long terme.

Règle d'or : définis 3 sorties AVANT d'entrer — 🛑 Stop (thèse fausse / le prix casse) · 🎯 Cible (thèse réalisée) · ⏳ Invalidation (le catalyseur n'arrive pas).

1. Sortie par la thèse (fondamental — le plus important)

Tu es entré sur un catalyseur avant le tweet. Tu sors quand :

  • Thèse réalisée — le mouvement a eu lieu, c'est sur tous les écrans. Le tweet The Assembly = souvent le signal de sortie, pas d'entrée. « Tu entres quand personne ne regarde, tu sors quand tout le monde arrive. »
  • Thèse cassée — contrat annulé, FDA refuse, l'investisseur stratégique se retire → sortie immédiate.

2. Sortie par catalyseur négatif (EDGAR — le pipeline les voit déjà)

SignalSourceSens
424B5 / offeringEDGARdilution → vendre
Form 4 — vente d'initiésEDGARles dirigeants cashent 🚩
Lock-up expiry (~180j)calculé (IPO + 180j)pression vendeuse datable
13F — investisseur réduitEDGARle smart money sort

3. Sortie par prix / technique (le timing — ici il compte)

  • RVOL qui s'effondre : le move s'est fait en RVOL élevé ; quand il retombe à ~1×, le carburant est parti → trim.
  • Exhaustion parabolique : spike + volume massif puis reversal → prends tes profits dans la force.
  • Cassure d'un niveau clé : clôture sous le breakout ou la MM20 → stop.

4. Sortie par risk management (la discipline)

  • Stop-loss fixé AVANT l'entrée (small-cap volatile : sous le support / −15 à −25 %). Le stop = survie.
  • Scale-out en R : encaisse à +1R, +2R, laisse courir le reste avec un trailing stop (ex. sous la MM10).
  • Sizing lié au stop : risque un % fixe du compte par trade.
Synthèse : Entrée → « jette le timing », tu entres sur le filing, avant le bruit. Sortie → le timing compte : tu sors quand le bruit arrive (tweet / RVOL spike / parabole) OU qu'un catalyseur négatif (424B5 / vente d'initiés) tombe.

11. Glossaire

RVOLRelative Volume = volume du jour / volume moyen. ≥ 2× = intérêt anormal.
FloatActions réellement disponibles à la négociation. Faible = mouvements amplifiés.
Short squeezeRachat forcé des vendeurs à découvert → flambée. Float faible + short élevé = terrain.
Lock-upPériode (~180j post-IPO) où les initiés ne peuvent pas vendre. Son expiration = pression vendeuse datable.
8-KDéclaration d'événement matériel à la SEC (sous 4 jours ouvrés).
424B5Prospectus d'émission = dilution (généralement baissier).
13F / Form 4Positions des institutions (13F) / transactions d'initiés (Form 4).
