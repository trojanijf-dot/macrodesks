Configuration alertes n8n
Colle ce nœud JavaScript dans ton workflow n8n après un Trigger Cron quotidien (ex: 7h00 NCT chaque matin). Il génère les emails d'alerte pour les factures en retard ou proches de l'échéance.
Nœud JS — Alertes Factures (à coller dans n8n)
Architecture workflow n8n recommandée
1. Trigger Cron → tous les jours à 7h00 NCT (UTC+11)
2. Code JS "Alertes Factures" → lit les données JSON + calcule les alertes
3. IF → alertes détectées ? (si 0 alertes → stop)
4. Formater Email HTML → template email professionnel
5. SMTP → envoyer à jftrojani@outlook.com
⚠ Pour que n8n lise les factures, tu dois exporter le JSON depuis l'onglet "Nouvelle facture" et le stocker dans une variable de workflow statique via $getWorkflowStaticData('global').invoices
